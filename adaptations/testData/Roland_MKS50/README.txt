Selected test fixtures from MKS50_Alpha_Juno_Collection-A.zip.

Only a small subset is included for automated tests:
- FACTORYA.SYX is a stock MKS-50 tone BLD bank.
- GLORIOUS_ALPHA.SYX, ALIEN_MKS50_aJUNO1-2.SYX, MIKEJUNO.SYX, and MISC1_MKS50_aJUNO1-2.SYX are Alpha/Juno-labeled tone BLD banks that use the same MKS-50 tone format.
- EZBANK1.SYX is a tone BLD bank with an unterminated trailing SysEx fragment.
- x-PatchParameters-MKS50-FACTORYA.SYX is a patch-parameter BLD bank.
- x-ChordMemory-MKS50.SYX is a chord-memory BLD file.
